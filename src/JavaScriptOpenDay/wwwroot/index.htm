About the JavaScript Open Day

  • Full Day Event

    JavaScript Open day is a full day event all about JavaScript and the Web presented by Microsoft Edge. Registration opens at 8.30am, keynote kicks off at 9.30am and sessions finish at 4.30pm.

  • World Class Speakers

    We will have 7 expert speakers from the Web Community speaking about all aspects of JavaScript development, from ES6 and cutting edge client side development through to JS on the server.

  • See what’s new

    Expect Session to dive deep into detail on what's new in JavaScript and Web Development

  • Lightning Talks

    Hear lightning talks from fresh new UK Web talent. Ideas you won't of heard from exciting new speakers.

What you’ll learn

What’s new from the Microsoft Edge team: EdgeHTML, Chakra, and Modern.IE

How to build blazing-fast sites and apps using JavaScript without breaking the web

New easy tools to build and submit cross-platform apps for multiple stores

How to create rich multimedia experiences with WebGL and Web Audio
